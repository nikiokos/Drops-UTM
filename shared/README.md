# @drops-utm/shared

Shared TypeScript type definitions for the DROPS UTM system.

## Structure

- `src/enums/` - Enum definitions for statuses, types, and roles
- `src/types/` - TypeScript interfaces for all entities

## Usage

```typescript
import { Hub, HubStatus, Flight, FlightStatus } from '@drops-utm/shared';
```

## Build

```bash
npm run build
```
